# Indica Otaku

**Indica Otaku** é uma aplicação web que permite aos usuários receber recomendações aleatórias de mangas e manhuas com base em diferentes categorias. Os usuários também podem pesquisar por títulos, autores, ou gêneros de mangas e manhuas específicos. Este projeto foi desenvolvido como parte da **Imersão Alura**, com o objetivo de praticar HTML, CSS, e JavaScript, criando uma interface interativa e dinâmica.

## Funcionalidades

- **Recomendações Aleatórias:** Os usuários podem selecionar uma categoria e receber duas recomendações aleatórias de mangas/manhuas daquela categoria.
- **Pesquisa de Mangas/Manhuas:** Possibilidade de pesquisar por títulos, autores ou gêneros específicos.
- **Design Responsivo:** O layout é otimizado para dispositivos móveis, oferecendo uma experiência de usuário fluida em diferentes tamanhos de tela.

## Tecnologias Utilizadas

- **HTML5**: Estrutura semântica do site.
- **CSS3**: Estilização, incluindo layout responsivo e efeitos de hover.
- **JavaScript (ES6+)**: Manipulação do DOM para recomendações dinâmicas e busca em tempo real.
- **Google Fonts**: Tipografia utilizada no projeto.
- **Imagens e Ícones**: Imagens de fundo, ícones de redes sociais e logotipos utilizados no rodapé.

##Como Usar
Na página principal, escolha uma categoria clicando em um dos botões disponíveis. Após selecionar uma categoria, clique no botão "INDICAR" para gerar duas recomendações de mangas ou manhuas.
Use a barra de pesquisa para encontrar um título específico, seja pelo nome do manga/manhua, autor ou gênero.
No rodapé, você pode encontrar links para as minhas redes sociais e o site da Alura, onde participei da Imersão.
